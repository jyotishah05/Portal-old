# OldAgeCare-Portal

## 📖 Project Overview

OldAgeCare-Portal is a responsive multi-page web application designed to help families arrange care and support for elderly parents. The idea behind this project is to provide a simple platform where children living away from their parents can explore elderly care services and stay connected with their well-being.

This project was developed to strengthen frontend web development skills while addressing a meaningful real-world problem through an easy-to-use and responsive interface.
